## Уровень 4 — Хардкор  
**Задача**  
Сделать защищённый ввод с повтором при ошибке + красивую карточку в конце

**Нужные методы:**
- `ReadString(prompt, minLength)`  
- `ReadInt(prompt, min, max)`  
- `ReadDouble(prompt, min, max)`

**Что запрашивать:**
- Имя (мин 2 символа)  
- Возраст (18–100)  
- Рост см (100–250)  
- Вес кг (30–300)
