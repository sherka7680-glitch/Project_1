День 1 — Переменные, типы данных, ввод/вывод, арифметика
Уровень 1 — Beginner
Напиши программу, которая:

Запрашивает у пользователя имя (строка)
Запрашивает возраст (целое число)
Выводит приветствие в формате:
Привет, [имя]! Тебе [возраст] лет.
Вычисляет и выводит, сколько лет будет через 5 лет.

Пример вывода:
textВведите ваше имя: Шерка  
Введите ваш возраст: 25  
Привет, Шерка! Тебе 25 лет.  
Через 5 лет тебе будет 30 лет.
Уровень 2 — Средний
Расширь предыдущую программу:

Запроси ещё рост в сантиметрах (целое число)
Запроси вес в килограммах (вещественное число, double)
Вычисли и выведи индекс массы тела (ИМТ) по формуле:
ИМТ = вес / (рост_в_метрах * рост_в_метрах)
Выведи результат с 2 знаками после запятой
Добавь простую рекомендацию:
ИМТ < 18.5 → "Недостаточный вес"
18.5–24.9 → "Норма"
25–29.9 → "Избыточный вес"
≥ 30 → "Ожирение"


Уровень 3 — Сложный
Создай консольный "калькулятор возраста в разных единицах":

Запроси возраст в годах (int)
Вычисли и выведи:
Возраст в месяцах
Возраст в неделях
Возраст в днях
Возраст в часах
Возраст в минутах
Возраст в секундах

Используй const для количества дней в году (365.25), часов в сутках (24) и т.д.
Форматируй большие числа с разделителями тысяч (например, 1_234_567) — используй .ToString("N0") или интерполяцию с :N0

Уровень 4 — Хардкор
Сделай "валидированный вводчик персональных данных" с обработкой ошибок:

Создай отдельный метод ReadInt(string prompt, int min, int max) — запрашивает целое число, повторяет ввод при ошибке или выходе за диапазон
Создай метод ReadDouble(string prompt, double min, double max) — аналогично для double
Создай метод ReadString(string prompt, int minLength) — строка не короче minLength
В Main используй эти методы для ввода:
Имя (мин. 2 символа)
Возраст (18–100)
Рост см (100–250)
Вес кг (30–300)

При любой ошибке ввода — выводить понятное сообщение и просить ввести заново (не крашить программу)
После всех данных — вывести всю информацию красиво (в виде "карточки") + посчитать ИМТ + рекомендацию
